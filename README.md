# SmartURL

A production-ready URL Shortener built with Java and Spring Boot.

## Tech Stack

- Java 21
- Spring Boot
- PostgreSQL
- Redis (Upcoming)
- Kafka (Upcoming)
- Docker
- Swagger
- JUnit

## Current Features

- URL Shortening
- URL Redirection
- Swagger API
- Validation
- Global Exception Handling
